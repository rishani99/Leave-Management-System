# Leave Management System (Java + MySQL)

This is a Leave Management System built using **Java Swing**, **OOP**, and **MySQL**.

## Features
- Employee Sign-Up & Login
- HR Sign-Up & Login
- Admin Sign-Up & Login
- Leave Management
- Dashboard UIs
- Database Connectivity (JDBC)

## Technologies
- Java (Swing UI)
- MySQL / XAMPP
- JDBC
- MVC Structure

## Setup Guide
1. Clone repository
2. Import project in  Eclipse 
3. Import `dbconnection.sql` to MySQL
4. Update DB username & password in `DBConnection.java`
5. Run `HomeUI.java`
